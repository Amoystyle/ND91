【friDay助手】WiFi連結幫助

1.如何透過WiFi網路與PC端的friDay助手連結？

  • 點擊手機上的【立即掃描】按钮，會出现QR CODE掃描器；

  • 打開電腦上的friDay助手，在未連結界面找到wifi连連結的QR CODE，如下圖。將手機的QR CODE 掃描器對準電腦上助手的QR CODE，即可自動連結。

2.按提示操作後仍連不上WiFi，是什麼原因？

  • 請檢查您的電腦是否和目前連結的手機處於同一個區網，如果不是，也無法透過WiFi連結；
  • 請檢查目前WiFi訊號是否穩定：在WiFi訊號不穩定的情况下，可能導致連結超時而最终導致無法連結的情況。
